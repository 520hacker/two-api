# two-api
这是一个带管理界面的AI对话转换和转发平台，参考了ONE-API，初期的业务逻辑和实现方式都会参考它。

## 如何使用：
把你的代理或者API host配置成https://twoapi.qiangtu.com

- 在token处填你生成的 令牌
- 在token页面也可以点击复制按钮把令牌复制之后用在别处
- 也可以直接选择直接点击打开 ChatNextWeb 或者 AMA 这样强大的客户端。
